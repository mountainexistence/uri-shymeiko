# Uri Shymeiko — сайт фотографа

Статический сайт-портфолио на собственном коде (HTML + CSS + JS), **без конструктора**.
Светлый минималистичный дизайн, фотография в центре внимания.

## Запуск
Открыть `index.html` в браузере (двойной клик). Сборка не нужна — чистая статика.

## Структура
- `index.html` — единственная страница (hero, о фотографе, портфолио, услуги, контакты)
- `css/style.css` — стили, адаптивность, анимации
- `js/main.js` — мобильное меню, лайтбокс (стрелки/Esc/клавиатура), reveal-анимации
- `assets/` — логотип (SVG), портрет, hero-фото
- `img/sport` · `img/bike` · `img/resort` — 55 работ портфолио по категориям

## Деплой
Любой статический хостинг: GitHub Pages, Netlify, Cloudflare Pages, nginx —
просто загрузить папку целиком.

## Внешние зависимости
- Шрифт Inter с Google Fonts (есть системный fallback)
- Ссылки на Instagram / VK
Всё остальное — локальное, работает офлайн.
